                                                   <h3 class="title-font">Allgemeine Vorhabenbeschreibung</h3>
                            <p>Die Röhm GmbH betreibt ein historisch gewachsenes System an Grundwasserbenutzungsanlagen mit nachfolgendem Rohrleitungssystem in Bächingen und dem benachbarten baden-württembergischen Sontheim zur Versorgung des Betriebs mit Kühlwasser, sowie zu Objektschutzmaßnahmen. Aus Sicht des vorsorgenden Grundwasserschutzes und mit Blick auf einen sorgsamen Umgang mit der Ressource Grundwasser bestand die Notwendigkeit das System grundlegend zu analysieren und wasserwirtschaftlich zu optimieren.<br/>Aufgrund der hydrogeologischen und hydraulischen örtlichen Gegebenheiten sind zum einen Grundwasserentnahmen zu Objektschutzgründen erforderlich, andererseits erfolgt an anderer Stelle eine Einleitung des Grundwassers aus dem Objektschutz zur Grundwasseranreicherung wiederum für die Entnahme zu Kühlzwecken. <br/>Aus dem bestehenden Brunnen 1 (Baden-Württemberg) soll künftig Grundwasser zu Objektschutzgründen entnommen werden. Ein Teil des dort entnommenen Grundwassers wird dem Brunnen 3 auf bayerischer Seite zugeführt, um dort das geringhöffige Grundwasservorkommen anzureichern. Aus diesem Brunnen wird Grundwasser entnommen, um den Kühlwasserkreislauf der Röhm GmbH zu speisen. Ein geringer Teil des Kühlwassers wird aus dem Kreislauf zu Prozesswasserzwecken abgezweigt und nach der Verwendung fachgerecht entsorgt. Der überwiegende Kühlwasserstrom wird nach der Verwendung zwei bestehenden Fischteichanlagen zugeleitet oder direkt Richtung Vorfluter Siechenbach bzw. Brenz abgeführt.<br/>Die Entnahme aus dem Brunnen 1 und die direkte Einleitung in die Vorfluter Siechenbach und Brenz werden wasserrechtlich auf baden-württembergischer Seite bearbeitet.</p>

    function getOSMLayer(attribution){
        var osmUrl='https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png';
        var osmAttrib='&copy; <a target="_blank" class="external-link" href="https://openstreetmap.org">OpenStreetMap</a> contributors';
        if(attribution){
            osmAttrib = osmAttrib + "" + attribution;
        }
        return new L.TileLayer(osmUrl, {attribution: osmAttrib});
    }

    function getWMSLayer(layerUrl, layerName, attribution){
        var osmAttrib='&copy; <a target="_blank" class="external-link" href="https://openstreetmap.org">OpenStreetMap</a> contributors';
        if(attribution){
            osmAttrib += attribution;
        }
        return new L.tileLayer.wms(layerUrl, {
            layers: layerName,
            attribution: osmAttrib
        });
    }

    function addLeafletMap(baselayers, bounds, latlng, zoom){
        return addLeafletMapWithId('map', baselayers, bounds, latlng, zoom);

    function addLeafletMapWithId(mapId, baselayers, bounds, latlng, zoom){
        var epsg = L.CRS.EPSG3857;
        if('3857'.length > 0) {
            epsg = L.CRS.EPSG3857;
        }
        var map = new L.Map(mapId, {

           layers: baselayers
        });
        if(bounds){
          map.fitBounds(bounds);
        } else if(latlng) {
          map.setView(latlng, zoom || 6);
        } else {
          map.setView(new L.LatLng(51.3, 10), 6);
        }
        return map;
    }

    function addLeafletHomeControl(map, title, position, icon, bounds, padding, fontSize){

            },

                var container = L.DomUtil.create('div', 'leaflet-control-home leaflet-bar');

                link.href = '#';
                if(padding){
                  link.style.padding = padding;
                }
                if(fontSize){
                  link.style.fontSize = fontSize;
                }
                link.title = title;
                L.DomEvent.addListener(link, 'click', this._homeClick, this);
                return container;
            },
            _homeClick: function(e) {
                L.DomEvent.stop(e);
                if(bounds){
                    var features = [];
                    map.eachLayer(function(layer) {
                        if(layer && layer.getBounds) {
                            features.push(layer);
                        }
                    })
                    if(features.length > 0) {
                        bounds = L.featureGroup(features).getBounds();
                    }
                    map.fitBounds(bounds);
                }
            }
        });
        map.addControl(new HomeControl({}));
    }
